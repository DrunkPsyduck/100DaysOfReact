# 100 Days Of React
## Objetivos
El objetivo durante estos 100 próximos días es recordar los fundamentos de React y profundizar en conocimientos mediante el desarrollo de diferentes aplicaciones de React.

Lista de Proyectos

1. clasificacion-pilotos


## Cuaderno De Bitácora
### Días:

1. Se empieza un curso de React en Udemy, se comienza repasando funciones modernas de JS e inicializando el proyecto de Citas-react
2. Se continua el desarrollo del proyecto añadiendo nuevos componente e instalando, probando Vite para el renderizado y usando jsx para los ficheros de React
3. Se añade funcionalidad al proyecto y se instala tailwindCSS
4. Se repasa el uso de los hooks y los props en React
5. Se continua añadiendo la funcionalidad de los componentes para editar y eliminar citas. Se termina el proyecto
6. Se  comienza un nuevo proyectos para añadir pilotos y repasar los conocimientos anteriores. Se sigue utilizando TailwindCSS pero no se usa Vite ni ficheros .jsx
